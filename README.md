# Unified-Interface-For-LIC

This repo contains a unified interface for training learned image compression (LIC) models.

Along with a unified interface, the following models have been integrated into the research platform for testing
and analysis:
- DCVC-RT I-frame 
- ELIC (Unofficial Implementation by JiangWeibeta)
- StableCodec
- LIC-HPCM
- JPEG-AI
- LIC-TCM
